# Demo Video Script

Use this when recording the mandatory demo video.

1. Start the project with `npm start`.
2. Open `http://localhost:3000`.
3. Show the dashboard stats, event cards, and role selector.
4. Switch between Admin, Photographer, Member, and Viewer to demonstrate access control.
5. Create a new event as Admin.
6. Upload multiple images as Photographer using the drag-and-drop upload panel.
7. Search by event name, tag, date, or uploader.
8. Like, favourite, comment, and download a media item.
9. Show the watermark prompt or downloaded watermarked image.
10. Use "Find my photos" to demonstrate the facial-recognition discovery flow.
11. Show the README, `required-deliverables/database/schema.sql`, and `required-deliverables/docs/architecture.md`.
